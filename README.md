# Programming on the Internet - My Third Website (Final Project)
Student project using C# and ASP.NET Core MVC.

Strona została przygotowana na zajęcia z przedmiotu Programowanie w Internecie. Strona wykorzystuje  HTML, CSS, JavaScript, Bootstrap (frontend) oraz język C# i ASP.NET MVC (backend). Dodatkowo strona jest w pełni responsywna oraz zawiera (na ten moment) dwie wersje językowe - polską i angielską.
